# Password Strength Checker (C)

A simple C program that checks the strength of a password using:
- Uppercase letters
- Lowercase letters
- Digits
- Special characters
- Length
- Entropy (randomness)

The program also gives suggestions to improve weak passwords.

## How to Run

1. Compile the program:
   gcc password_checker.c -o password_checker

2. Run it:
   ./password_checker   (Mac/Linux)
   password_checker     (Windows)

## Features
- Uses functions for cleaner code
- Calculates password entropy
- Rates password as WEAK, MEDIUM, STRONG, or VERY STRONG
- Provides improvement tips
